ChatGPT Atlas Opener

Open any webpage in the ChatGPT Atlas macOS app with a single click or keyboard shortcut.

Usage

  • Click the extension icon in your toolbar, or
  • Press ⌘ Command + ⇧ Shift + A

How It Works

This extension uses native messaging to communicate with macOS and launch URLs in ChatGPT Atlas using the open command. It bypasses Chrome's restrictions on custom URL schemes.

Requirements

  • macOS 14+ with Apple Silicon (M1 or better)
  • ChatGPT Atlas installed in Applications folder
  • Native messaging host installed (via install.sh)
